> Réalisation d’un site de sport d'une agence événementielles sous **React Js**

### 🏠 [Homepage](https://github.com/jbach-dev/event_sport)

## Install

```sh
npm install
```

## Usage

Start server && data

```sh
npm run data_server
```

New data

```sh
npm run data
```

Start application

```sh
npm start
```

## Spécificités

- Conception du site
- Présentation d’évènement avec filtre de recherche
